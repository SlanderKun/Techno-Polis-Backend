# Установка

1. Установка docker/ docker-compose

### В зависимости от версии docker-compose (старая версия -o)

2. Провести сборку

```bash
make build
```

3. Провести миграцию

```bash
make migrate
```

4. Запустить проект

```bash
make up
```
